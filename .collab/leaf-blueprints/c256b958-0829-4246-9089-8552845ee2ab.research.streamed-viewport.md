## What needs to be built

A new React component `StreamedViewport` that:
1. Gets the `WebSocketClient` for the session's specific server (via `getFrameClient(server)` from the sibling task `per-server-ws-resolver` → `ui/src/lib/serverFrameWs.ts`).
2. On mount, subscribes to channel `'browser:' + session` on that client.
3. On each `browser_frame` WS message matching the session, paints the base64 JPEG onto a `<canvas>` element.
4. Keeps the latest frame metadata (`offsetTop`, `pageScaleFactor`, `deviceWidth`, `deviceHeight`, `timestamp?`) in a `useRef` so L5 (input mapping) can read it at event time.
5. On unmount, unsubscribes from the channel and removes the message handler (but does NOT disconnect the shared client).

## WebSocketClient API (from `ui/src/lib/websocket.ts`)

Key methods used:
- `client.connect(): Promise<void>` — idempotent; call before subscribing.
- `client.subscribe(channel: string): void` — sends `{ type: 'subscribe', channel }` to server.
- `client.unsubscribe(channel: string): void` — sends `{ type: 'unsubscribe', channel }` to server.
- `client.onMessage(handler: (msg: WebSocketMessage) => void): { unsubscribe: () => void }` — returns a subscription handle with `.unsubscribe()`.
- `client.isConnected(): boolean` — current connection state.

`getWebSocketClient()` at `websocket.ts:379` is the existing same-origin singleton. The new `getFrameClient(server)` from `serverFrameWs.ts` wraps it for the C3 per-server binding.

## Effect logic (keyed on `[session, server?.id]`)

```ts
useEffect(() => {
  const client = getFrameClient(server);   // from serverFrameWs.ts
  let mounted = true;
  let sub: { unsubscribe(): void } | null = null;

  client.connect().then(() => {
    if (!mounted) return;
    client.subscribe('browser:' + session);
    sub = client.onMessage((msg) => {
      if (msg.type !== 'browser_frame') return;
      const frame = msg as unknown as { type: 'browser_frame'; session: string; data: string; meta: FrameMeta };
      if (frame.session !== session) return;
      lastMetaRef.current = frame.meta;
      paint(frame.data);
    });
  });

  return () => {
    mounted = false;
    sub?.unsubscribe();
    client.unsubscribe('browser:' + session);
    // Do NOT call client.disconnect() — it's a shared client.
  };
}, [session, server?.id]); // eslint-disable-line react-hooks/exhaustive-deps
```

## Paint function

MVP: one reused `Image` object (avoids allocation per frame):

```ts
const imgRef = useRef<HTMLImageElement | null>(null);
function paint(dataB64: string) {
  const canvas = canvasRef.current;
  if (!canvas) return;
  const ctx = canvas.getContext('2d');
  if (!ctx) return;
  if (!imgRef.current) imgRef.current = new Image();
  const img = imgRef.current;
  img.onload = () => {
    if (!canvasRef.current) return; // guard after unmount
    canvas.width = img.naturalWidth;
    canvas.height = img.naturalHeight;
    ctx.drawImage(img, 0, 0);
  };
  img.src = 'data:image/jpeg;base64,' + dataB64;
}
```

Notes:
- `canvas.width = img.naturalWidth` on each frame resizes the canvas if the frame dimensions change (e.g. window resize changes screencast resolution).
- Setting `.width` clears the canvas — that's fine because `drawImage` runs in `onload`.
- Guard `if (!canvasRef.current) return` inside `onload` to avoid painting after unmount.
- Blueprint notes an optional faster path with `createImageBitmap`; skip for MVP.

## Full component skeleton

```tsx
import { useRef, useEffect } from 'react';
import type { ServerInfo } from '@/contexts/ServerContext';
import { getFrameClient } from '@/lib/serverFrameWs';

export interface FrameMeta {
  offsetTop: number;
  pageScaleFactor: number;
  deviceWidth: number;
  deviceHeight: number;
  timestamp?: number;
}

export function StreamedViewport({
  session,
  server,
  metaRef,
}: {
  session: string;
  server?: ServerInfo;
  metaRef?: React.MutableRefObject<FrameMeta | null>;
}) {
  const canvasRef = useRef<HTMLCanvasElement>(null);
  const lastMetaRef = useRef<FrameMeta | null>(null);
  const imgRef = useRef<HTMLImageElement | null>(null);

  function paint(dataB64: string) {
    const canvas = canvasRef.current;
    if (!canvas) return;
    const ctx = canvas.getContext('2d');
    if (!ctx) return;
    if (!imgRef.current) imgRef.current = new Image();
    const img = imgRef.current;
    img.onload = () => {
      if (!canvasRef.current) return;
      canvas.width = img.naturalWidth;
      canvas.height = img.naturalHeight;
      ctx.drawImage(img, 0, 0);
    };
    img.src = 'data:image/jpeg;base64,' + dataB64;
  }

  useEffect(() => {
    const client = getFrameClient(server);
    let mounted = true;
    let sub: { unsubscribe(): void } | null = null;

    client.connect().then(() => {
      if (!mounted) return;
      client.subscribe('browser:' + session);
      sub = client.onMessage((msg) => {
        if (msg.type !== 'browser_frame') return;
        const frame = msg as unknown as { type: 'browser_frame'; session: string; data: string; meta: FrameMeta };
        if (frame.session !== session) return;
        lastMetaRef.current = frame.meta;
        if (metaRef) metaRef.current = frame.meta;
        paint(frame.data);
      });
    });

    return () => {
      mounted = false;
      sub?.unsubscribe();
      client.unsubscribe('browser:' + session);
    };
  }, [session, server?.id]); // eslint-disable-line react-hooks/exhaustive-deps

  return (
    <canvas
      ref={canvasRef}
      className="flex-1 w-full h-full object-contain bg-black"
    />
  );
}
```

## Key constraints from blueprint

- C3 (multi-server): Always use `getFrameClient(server)` — never import `getWebSocketClient()` directly in this component.
- `lastMetaRef` must be current at unmount time but only used by L5 at event time.
- No `disconnect()` on the shared client.
- `client.subscribe('browser:' + session)` — no `channel:` prefix (server adds it internally per `handler.ts:178`).
- No rAF loop, no `viewportRef`, no native overlay interaction.
- Channel name: `'browser:' + session` (matches `broadcastBrowserFrame` at `handler.ts:475-483` which targets `'browser:' + session`).
