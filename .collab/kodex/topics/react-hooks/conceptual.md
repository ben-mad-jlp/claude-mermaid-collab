# React Hooks

Custom React hooks used throughout the mermaid-collab UI for encapsulating reusable stateful logic.

## Available Hooks

| Hook | Purpose |
|------|--------|
| `useWebSocket` | WebSocket connection management |
| `useAutoSave` | Debounced content persistence |
| `useIsMobile` | Viewport size detection |
| `useTheme` | Dark/light mode management |
| `useDiagram` | Diagram CRUD operations |
| `useDocument` | Document CRUD operations |
| `useSession` | Session management |
| `useDataLoader` | Async data fetching |
| `useTerminalTabs` | Terminal tab management |
| `useSyncScroll` | Synchronized scrolling |
| `useEditorHistory` | Undo/redo for editors |
| `useExportDiagram` | Diagram export functionality |
