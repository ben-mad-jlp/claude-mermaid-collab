# GUI (Graphical User Interface)

The mermaid-collab GUI is a React-based web application providing visual tools for diagram editing, document management, and Claude collaboration.

## Access

The GUI is served by the HTTP server at `http://localhost:3737` and accessed via web browser.

## Main Views

1. **Editor View**: Split pane with code editor and live preview
2. **Document View**: Markdown editing with rendered preview
3. **Dashboard**: Session and project management
4. **Kodex**: Knowledge base browser

## Interactive Features

- Real-time diagram preview with zoom/pan controls
- Syntax highlighting in code editor
- AI UI overlays for Claude interactions
- Toast notifications for system events
- Theme switching (light/dark)
