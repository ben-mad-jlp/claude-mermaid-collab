# Responsive Design

The UI adapts to different viewport sizes with a mobile-first approach, switching between desktop and mobile layouts at the 640px breakpoint.

## Breakpoint Strategy

- **Mobile**: < 640px width - Tab-based navigation, full-screen views
- **Desktop**: >= 640px width - Multi-panel layout with sidebar

## Layout Differences

### Desktop
- Fixed sidebar with item list
- Split pane for editor + chat panel
- Header with full session controls
- Resizable panels

### Mobile
- Bottom tab navigation (Preview, Chat, Terminal)
- Full-screen content areas
- Compact header with hamburger menu
- Slide-up item drawer
- Touch-optimized interactions
