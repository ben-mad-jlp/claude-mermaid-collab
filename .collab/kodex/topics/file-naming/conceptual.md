# File Naming Conventions

Consistent naming patterns used across the codebase.

## Summary

| Category | Convention | Example |
|----------|-----------|--------|
| Backend files | kebab-case | `diagram-manager.ts` |
| React components | PascalCase | `CodeBlock.tsx` |
| Component directories | kebab-case | `ai-ui/`, `chat-drawer/` |
| Test files | `.test.ts` suffix | `api.test.ts` |
| Test directories | `__tests__/` | `/src/services/__tests__/` |
| Skills | kebab-case dir | `test-driven-development/` |
| Skill files | UPPERCASE | `SKILL.md` |
