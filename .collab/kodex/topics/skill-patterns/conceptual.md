# Skill Patterns

Skills are Claude Code workflow definitions stored as markdown files with YAML frontmatter.

## Directory Structure

- Location: `/skills/`
- 38+ skills organized in separate directories
- Each directory contains one `SKILL.md` file
- Naming convention: kebab-case (e.g., `test-driven-development/`)

## Skill Categories

| Category | Skills | Purpose |
|----------|--------|--------|
| Collab | 5 | Session management |
| Brainstorming | 4 | Design refinement phases |
| Rough Draft | 4 | Progressive implementation |
| Execution | 3+ | Task execution |
| Kodex | 7 | Knowledge base management |
| Utility | 4+ | Design, planning, review |
